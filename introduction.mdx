## Getting Started

Welcome to Aaron AI, a powerful platform for organizing and managing project documentation. Aaron AI provides a centralized space where teams can create, collaborate on, and maintain comprehensive documentation for their projects. Whether you're building APIs, developing software, or documenting processes, Aaron AI offers the tools you need to keep your documentation accurate, up-to-date, and accessible.

## Key Features

Aaron AI stands out with its intuitive interface and robust feature set. You can create structured documentation with rich formatting, embed code examples, and organize content into logical sections. The platform supports multiple formats, including Markdown, and provides version control to track changes over time.

<Tabs>
  <Tab title="For Developers" icon="code">
    <Callout kind="info">Developers can use Aaron AI to document APIs, SDKs, and codebases with syntax highlighting and interactive examples.</Callout>
    <Steps>
      <Step title="Install SDK" icon="download">
        Install the Aaron AI SDK using npm:
        ````bash
        npm install aaron-ai-sdk
        ````
      </Step>
      <Step title="Authenticate" icon="key">
        Set up authentication with your API key:
        ````javascript
const client = new AaronAIClient({
  apiKey: process.env.AARON_API_KEY
});
        ````
      </Step>
      <Step title="Create Documentation" icon="file-text">
        Start documenting your project:
        ````javascript
await client.documents.create({
  title: 'API Reference',
  content: '# My API\n\nThis is my API documentation.'
});
        ````
      </Step>
    </Steps>
  </Tab>
